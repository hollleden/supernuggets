# supernuggets

A personal "second brain." A Telegram bot ingests text, images, voice notes, videos, and links — AI transcribes, classifies, tags, and files each item into one of 12 folders. A web app is the visual front door.

## Stack

- **Web:** Next.js (App Router), TypeScript, Tailwind, shadcn/ui
- **Bot:** Python, aiogram v3
- **Database:** Supabase (Postgres)
- **AI:** Anthropic Claude + OpenAI Whisper
- **Media:** yt-dlp, trafilatura, Pillow

## Folders

```
Grow · Leisure · Health · Creativity · Money · Work
Curation · Personal · Beauty · Food · Travel · Sport
```
